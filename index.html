Etapa basica

HTML

HTML es el lenguaje utilizado para crear páginas web, proporcionando la estructura y el contenido visual a través de etiquetas y elementos. Es esencial en el desarrollo web y se complementa con otros lenguajes como CSS y JavaScript.

CSS

CSS, siglas en inglés de Cascading Style Sheets (Hojas de Estilo en Cascada), es un lenguaje utilizado para definir el aspecto y el formato visual de las páginas web escritas en HTML. Permice especiricar como se presencan los elementos HTML, como el color, la fuente, el tamaño y la disposición en la página. CSS se utiliza para mejorar la apariencia y la usabilidad de los sitios web, separando la estructura del contenido HTML de su presentación visual.

JavaScript

JavaScript es un lenguaje de programación ampllamente ullizado en el desarrollo weo para crear interactividad en las páginas. Permite agregar funcionalidades dinámicas y JS comportamientos a los sitios web, como animaciones, validaciones de formularios, manipulación del DOM (Document Object Model) y comunicación con servidores. JavaScript es compatible con todos los navegadores web modernos y es fundamental en la construcción de aplicaciones web interactivas y dinámicas.

SQL

SQL es un lenguaje para gestionar y manipular bases de datos relacionales. Permite realizar consultas y operaciones como inserciones, actualizaciones y eliminaciones de datos. Es esencial en el desarrollo de aplicaciones que requieren almacenamiento y gestión de datos estucluiduos.

MongoDB

MongoDB es una base de datos NoSQL que utiliza documentos similares a JSON para almacenar datos de manera flexible y escalable, siendo ideal para aplicaciones wed y moviles.
